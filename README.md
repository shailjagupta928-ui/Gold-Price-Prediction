Gold Price Prediction using Machine Learning
📌 Overview

This project focuses on predicting gold prices using the Random Forest Regressor, a powerful ensemble learning technique. The model analyzes historical data and learns patterns to make accurate future price predictions.

🚀 Features
Predict gold prices based on historical data
Uses only Random Forest Regressor for robust performance
Handles non-linear relationships effectively
Reduces overfitting using ensemble learning
Easy-to-use and deployable
🧠 Model Used
Random Forest Regressor
Combines multiple decision trees
Uses averaging to improve accuracy
Works well with complex datasets
Less prone to overfitting compared to single models
